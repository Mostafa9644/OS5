.git :

سوال 3 : دستوراتی معرفی کنید تا بتوان ابتدا یک دایرکتوری در سیستم بسازیم و سپس به گیت هاب وصل کنیم

سپس این دستورا ت رو وارد میکنیم.

سپس در سایت گیت هاب میرویم و ریپازیتوری جدیدی با نام
